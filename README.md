# ATM-Management-System
A Java-based ATM Management System developed using JDBC and PostgreSQL. The application allows users to perform banking operations through a console interface.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Display Account Details
- Update Account Details
- Transfer Amount
- Exit

## Technologies Used
- Java
- JDBC
- PostgreSQL
- Maven
- Eclipse IDE

## Project Structure
- ATMMain.java - Main class
- AtmImplementation.java - Business logic
- AtmHiding.java - Interface
- DatabaseConnection.java - Database connectivity
- InvalidDataException.java - Custom exception

## Prerequisites
- Java 8 or later
- PostgreSQL
- Maven
- Eclipse IDE

## How to Run
1. Clone this repository.
2. Create the required database and tables in PostgreSQL.
3. Update the database username and password in `DatabaseConnection.java`.
4. Run `ATMMain.java`.
